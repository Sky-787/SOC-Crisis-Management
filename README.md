# SOC-Crisis-Management
Sistema asimétrico - gestión - crisis - ciberseguridad - Dos jugadores - tiempo real. Un jugador actúa como **Monitor** (observa el estado del sistema bajo ataque) y el otro como **Técnico** (ejecuta acciones de mitigación). Ambos se comunican verbalmente y coordinan sus acciones para detener un ataque de ransomware antes de que el sistema colapse.
